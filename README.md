# Automatizovaný návrh zubového čerpadla

Webový nástroj pro komplexní inženýrský návrh geometrie zubových čerpadel a automatický výběr vhodného pohonu.

## Hlavní funkce
* **Výpočet geometrie**: Výpočet modulu (Řada I), počtu zubů, roztečných a hlavových průměrů.
* **Konstrukční kontroly**: Automatické hlídání obvodové rychlosti (≤ 4.0 m/s), poměru lambda a tolerance průtoku.
* **Výkonové parametry**: Výpočet potřebného příkonu čerpadla s ohledem na celkovou účinnost a bezpečnostní přirážku (1.4x).
* **Volba motoru**: Přímé doporučení konkrétních elektromotorů VYBO Electric z aktuálního ceníku (4, 6 a 8-pólové verze) včetně odkazů na e-shop.
